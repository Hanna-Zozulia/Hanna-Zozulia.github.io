# Software Explorer

## Описание проекта
Это веб-приложение представляет собой интерактивные обучающие карточки для изучения программного обеспечения. Пользователь может:
- Просматривать карточки с названием и типом ПО.
- Переворачивать карточки для изучения информации.
- Помечать карточки как выученные.
- Перетаскивать карточки в соответствующие зоны во время теста.
- Проверять свои знания с помощью интерактивного теста.

Приложение разработано с использованием **HTML, CSS и JavaScript** и демонстрирует навыки работы с DOM, событиями, drag-and-drop и управлением состоянием приложения.

---

## Особенности
1. **Карточки для обучения**  
   - Карточки можно переворачивать, чтобы увидеть дополнительную информацию.  
   - Карточки можно скрывать или помечать как выученные.  

2. **Перетаскиваемый тест**  
   - Перетаскивание карточек в соответствующие категории.  
   - Автоматическая проверка результатов.  
   - Подсветка правильных и неправильных ответов.  

3. **Управление состоянием**  
   - Сохраняются выученные карточки.  
   - Возможность возвращать карточки обратно в обучение из выученных.  

4. **Интерактивный интерфейс**  
   - Кнопки «Начать тест», «Перемешать», «Показать выученные» и «Назад».  
   - Адаптивное отображение карточек в контейнере.

---

## Используемые технологии
- HTML5
- CSS3 (Flexbox, кнопки с иконками через Bootstrap Icons)
- JavaScript
- Drag & Drop API

---

## Инструкция по запуску
1. Скачайте проект или клонируйте репозиторий.  
2. Откройте `index.html` в браузере.  
3. Используйте карточки для изучения и тестирования своих знаний.

---

## Цель проекта
- Демонстрация навыков работы с DOM и событиями.
- Реализация интерактивного пользовательского интерфейса.
